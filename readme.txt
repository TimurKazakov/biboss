Добрый день.
Данный проект сверстан по документу расположенному по адресу https://yadi.sk/i/6YE6PjJw3EqiJn
Валидность верстки проверялась по PerfectPixel by WellDoneCode, скриншоты расположены в папках img.
Технологии проекта:
Использовались Adobe Photoshop, Chrome, Opera, Safari, Firefox, WebStorm, Git
HTML - нативный
СSS - препроцессор Less, JQueryUI
JS - нативный, JQuery, сторонние скрипты, Ajax(имитация запроса на сервер, через файл data.json)

Проект собран на Webpack.
В ходе сборки использовались:
Babel, postcss, autoprefixer, cssnano

Для запуска сборки команда: npm run build
Для запуска сервера команда: npm run dev

Собранный проект находится в папке dist.
